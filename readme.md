\#  Чат-бот Калькулятор



Простой чат-бот на чистом JavaScript для вычисления арифметических выражений. Работает полностью в браузере, без серверной части.



\##  Возможности



\-  Красивый адаптивный интерфейс

\-  Общение с ботом в реальном времени

\-  Выполнение арифметических операций: +, -, \*, /

\-  Анимация печати (троеточие)

\-  Полностью адаптивен под все устройства

\-  Интуитивно понятное управление
